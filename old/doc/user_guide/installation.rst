Installation
============

You can install Deep Logic along with all its dependencies from
`PyPI <https://pypi.org/project/deep-logic/>`__:

.. code:: bash

    pip install -r requirements.txt deep_logic

or from source code:

.. code:: bash

    git clone https://github.com/pietrobarbiero/deep-logic.git
    cd ./deep_logic
    pip install -r requirements.txt .

Deep Logic is compatible with Python 3.7 and above.
